# INA228 Register Map

| Addr | Register | Bits | R/W | Description |
|---|---|---|---|---|
| 0x00 | CONFIG | 16 | R/W | Global configuration (reset, conversion delay, temp compensation) |
| 0x01 | ADC_CONFIG | 16 | R/W | ADC configuration: sampling time, averaging, mode |
| 0x02 | SHUNT_CAL | 16 | R/W | Calibration register, used for current calculation |
| 0x03 | SHUNT_TEMPCO | 16 | R/W | Shunt temperature coefficient |
| 0x04 | VSHUNT | 24 | R | Shunt voltage |
| 0x05 | VBUS | 24 | R | Bus voltage |
| 0x06 | DIETEMP | 16 | R | Die temperature |
| 0x07 | CURRENT | 24 | R | Current |
| 0x08 | POWER | 24 | R | Power |
| 0x09 | ENERGY | 40 | R | Accumulated energy |
| 0x0A | CHARGE | 40 | R | Accumulated charge |
| 0x0B | DIAG_ALRT | 16 | R/W | Diagnostic / Alert |
| 0x0C | SOVL | 16 | R/W | Shunt over-voltage limit |
| 0x0D | SUVL | 16 | R/W | Shunt under-voltage limit |
| 0x0E | BOVL | 16 | R/W | Bus over-voltage limit |
| 0x0F | BUVL | 16 | R/W | Bus under-voltage limit |
| 0x10 | TEMP_LIMIT | 16 | R/W | Temperature limit |
| 0x11 | PWR_LIMIT | 16 | R/W | Power limit |
| 0x3E | MANUFACTURER_ID | 16 | R | Manufacturer ID |
| 0x3F | DEVICE_ID | 16 | R | Device ID |

# Key Register Details

## 1️⃣ CONFIG (0x00)

```
15    RST       Reset device
14    RESERVED
13    TEMPCOMP  Enable shunt temp compensation
12:6  CONVDLY   Conversion delay
5:0   RESERVED
```

---

## 2️⃣ ADC_CONFIG (0x01)

Core sampling configuration:

```
15:12 AVG       Averaging (1..1024)
11:9  VBUSCT    Bus voltage conversion time
8:6   VSHCT     Shunt conversion time
5:3   VTCT      Temperature conversion time
2:0   MODE      Operating mode
```

**MODE:**

| Value | Mode |
|---|---|
| 000 | Shutdown |
| 001 | Shunt triggered |
| 010 | Bus triggered |
| 011 | Shunt + Bus triggered |
| 101 | Shunt continuous |
| 110 | Bus continuous |
| 111 | Shunt + Bus continuous |

---

## 3️⃣ SHUNT_CAL (0x02)

Used for calculating current:

```
CURRENT = VSHUNT / RSHUNT
```

Drivers typically set the **current LSB scaling** via this register.

---

## 4️⃣ Measurement Registers

### VSHUNT (0x04)

```
24-bit signed
LSB = 312.5 nV (range ±163.84 mV)
```

---

### VBUS (0x05)

```
24-bit unsigned
LSB = 195.3125 µV
Range = 0 - 85V
```

---

### CURRENT (0x07)

```
24-bit signed
Scaled by SHUNT_CAL
```

---

### POWER (0x08)

```
POWER = CURRENT × VBUS
```

---

### ENERGY (0x09)

```
40-bit accumulator
Energy = Σ Power × Time
```

---

### CHARGE (0x0A)

```
40-bit accumulator
Charge = Σ Current × Time
```

---

# I²C Access Flow

**Read Register:**

```
START
ADDR + W
REGISTER_ADDR
RESTART
ADDR + R
READ MSB
READ LSB
STOP
```

Registers are read/written **MSB first**.

---

# Most Common Registers for Drivers

When acting as a power monitor, only the following registers are usually needed:

```
0x01 ADC_CONFIG
0x02 SHUNT_CAL
0x04 VSHUNT
0x05 VBUS
0x06 DIETEMP
0x07 CURRENT
0x08 POWER
```

`ENERGY` and `CHARGE` are generally only used for **battery coulomb counting**.
